SateliteTracker
===============

A project for Web API to predict a satelite orbit using C++ Library (OrbitTools).

Plan, Description: http://tsujimotter.info/2013/01/02/iss-api/

Web API example: http://tsujimotter.info/webgl/orbitjsonp.cgi
